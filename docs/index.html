ClaudeBrew

Brewing answers while you brew coffee

Answer Claude Code prompts from your phone over local WiFi. No cloud, no accounts — just two devices on the same network.

View on GitHub

How it works

1

Start the daemon

Run the daemon on your Mac. Hooks install automatically and a QR code is printed in the terminal.

2

Scan & connect

Open the mobile app and scan the QR code, or manually enter the IP address and PIN.

3

Requests stream live

Claude Code tool use requests and terminal output stream to your phone in real time.

4

Approve from anywhere

Approve or deny permissions with a tap — from the couch, the kitchen, wherever you are.

Quick setup

Daemon (Mac)

# Install and start
cd apps/daemon
npm install
npm run build
npm start

Mobile (Expo)

# Start the dev server
cd apps/mobile
npm install
npm start

No configuration needed — hooks install automatically on first run.
